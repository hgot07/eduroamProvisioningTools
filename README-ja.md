# eduroamProvisioningTools
Wi-Fiプロファイルを用いて、eduroamの接続設定を容易にするための、プロファイル生成ツールとコード例です。機関の構成員(教職員や学生)向けのウェブサイトで、ログインが必要な場所に組み込むことで、プロファイル発行サービスを提供できるようになります。

標準では、eduroamのアカウントを使ってOpenRoamingに接続する方法に対応していませんが、コードの変更は容易でしょう。

認証方式は、EAP-TTLSとEAP-TLSのみ対応しています。
いくつかのOSは、EAP-TLSをサポートしていないことがあります。

**プロファイルを用いることで、利用者はID・パスワードを手入力する必要がなくなり、容易かつ正確なeduroam接続設定が可能になります。**
 
**偽基地局を回避するための「サーバ認証」の機能を、確実に設定することができます。**  以下のような問題も減らすことができます。
- 学内では接続できるのによそでは接続できない、または、この逆のケース。(手動設定でTOFU (Trust on First Use)が使われ、端末がWPA3専用モードに設定されることがあり、PMF非対応のWPA2の基地局に接続できなくなる)
- 認証サーバの証明書が更新されると、ネットワーク接続に失敗するようになり、再設定が必要になる。 (手動設定のTOFUでは、Root CAではなく、サーバ証明書のハッシュ値が端末に記録される。このため、証明書が変更されるとサーバ認証に失敗するようになる)
